## GitHub Pages Deployment

This site is configured for GitHub Pages. To deploy:

1. Go to repository Settings
2. Navigate to Pages section
3. Under "Source", select the branch to deploy (main)
4. The site will be available at `https://fxstudios.github.io`

## Local Development

Simply open `index.html` in a browser to preview the site locally.
